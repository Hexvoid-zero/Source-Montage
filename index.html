One command connects any agent

Launch SourceMontage.exe and the MCP server is live on your machine. Want a remote agent to reach it? Run serve-public.ps1 — it prints a free public https URL. Paste it below.

  • ✓Public when you want — free Cloudflare quick tunnel
  • ✓Works everywhere — Claude Code, Cursor, any MCP client
